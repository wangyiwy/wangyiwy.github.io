2019年终总结

发表于 2019-12-30 分类于 年终总结 阅读次数：

21世纪10年代最后一个冬天很快就过去了，毕业之后感觉每一年过的都特别的快。这是我第一次写年终总结，我想记录一下这一年发生的重要的事情。

  • 1月 年初的时候我开始转向 Java 后端开发，这是我工作以来一直想做的事情，对于我的职业生涯来说是一个重要的转折。

  • 2月 读完了《自己动手写Java虚拟机》 这本书，这也是今年读的唯一一本书。我觉得这本书对于我理解 JVM 有很大的帮助。

  • 4月 买了人生的第一套房子，虽然位置偏房子也不大，但我还是觉得比较满意，不用再对房价感到焦虑。

  • 7月 我写的第一个网站oktools上线。这是一个程序员常用的工具站，原型、UI、前端、后端都由自己完成的感觉十分有成就感，也学到了不少网站运营、SEO等方面的知识。目前日IP在300-400左右，这个项目只是练手和满足自己的需求，我已经觉得比较满意了。

  • 8月 我开始写我一直想做的一个项目-一个类似 V2EX 的论坛。目前项目只是完成了后端大部分功能，前端基本上还未开始。原因在于我想把交互和UI做到比较高的水平，但是自己本身对于前端不是很熟悉，UE/UI水平也十分有限，所以我已经很久没有写这个项目了。等接下来一段时间重新开始吧。我喜欢论坛这种交流方式，做一个论坛是我埋藏了多年的心愿，我会坚持下去。

  • 9月 毕业以来，第一次在一家公司待一年以上，这一年加的班比过去3年加的还多，十分怀恋不加班的日志。

  • 10月 我开始还我的房贷了，接下来的30年每个月都要还这笔贷款，让我不敢轻易离职。

  • 11月 我写完了第二个网站，我暂时叫它Whisper。灵感来源于豆瓣电影的年度榜单。我想收集让我深受感触的电影台词。目前写完了代码但还未收集数据。

时间快得让我觉得焦虑，难以想象，2009年已经是十年前。十年前我没有做到的事情，十年后我也没有做到。人生呐，为何有那么多遗憾。

最近几年跨年的时候我都在听这首歌，今年也将如此。

Why do some things stay the same when some don’t?

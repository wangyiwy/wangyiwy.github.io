重新开始写博客

发表于 2019-12-16 更新于 2019-12-20 阅读次数：

已经差不多两年没有写博客了，Github上的博客仓库也早被我删除了，wangyi.blog这个域名去年到期了也没有续费。最近发现这个域名还可以注册就又买了回来，打算重新开始搭建个博客吧。还是使用免费的Hexo + Github Pages的方式。

本来想着自己写一套Hexo的主题，但是自己时间不是很多，另外对于博客来内容才是最重要的，所以还是选用了之前的Next主题。简单配置了下，简洁又好看，如果自己来实现确实做不到这么精致的UI。

由于我可能会同时使用家里的电脑和工作的电脑写作，Hexo在多台电脑之间同步还是有点麻烦。目前我的解决方案是将_config.yml、themes\next_config.yml和source目录单独上传到一个Github private仓库。本来想着使用分支的方式管理，但是配置文件中有一些敏感的key，还是放到私有仓库安全一些。

对于我来说，写博客是一件比较能长久坚持的事情。因为觉得自己写的东西在互联网上已经很容易找到，但我现在想法有所改变，博客不一定是写给其他人看的。学习新的知识的时候，自己写一遍理解会加深很多的，同时写博客也能提升归纳总结的能力。
